A report from the course DA377B Software Development for the Web

S01

Web development is something I personally do not have much experience with, unless one counts setting up a page on Geocities in the late 1990s. Developing a webpage was part of a project we completed as a group in Software Development 2, but this was not a subgroup I had any involvement with.

I did use GitHub as part of that project and other projects and have some familiarity with using push, pull, commit and merging. I much prefer to use the GitHub desktop application rather than GitBash but I understand why it is important to learn the fundamental commands used in GitBash. I would note too that GitBash is probably the more powerful tool, too, as when something goes awry using the desktop application – usually when merging – we resort to using GitBash.

TIL:

  • Using text editors such as Visual Studio Code and Atom
  • Using links in a text editor
  • Forking a branch on Git

S02

Getting into the HTML and CSS side of web development is, I must admit, more enjoyable than I would have thought. Traditionally I have struggled with Java programming, and here we have a platform that incorporates the principles of object-oriented programming... but this seems more intuitive somehow. This could also be attributed to the extensive tutorials freely available online.

Working exclusively in HTML and CSS thus far, I understand this is what's referred to as 'front-end' development, or development for those with a slightly more artistic persuasion than a mechanical one. In that, one observation I would make is that front-end development can be a deceptive time-sink: when it came time to play around with font selection, I lost a whole hour just browsing fonts on Google. Same thing when I started looking at background colours.

TIL:

  • Fundamentals of coding in HTML
  • Fundamentals of coding using CSS
  • Not spending an hour looking at fonts

S03

This project saw us bringing together the three elements of HTML, CSS and javascript, and much more than that. This, being only the third assignment, provides a fairly steep learning curve in fetching from a remote source and incorporating events in javascript.
